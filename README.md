# network-logs-platform

A modular observability platform for collecting, normalizing, processing and storing Syslog and NetFlow logs using Vector, Kafka and OpenSearch.

Designed around the Elastic Common Schema (ECS), Data Streams and lifecycle policies, it provides a scalable foundation for network observability, security monitoring and long-term log retention.


___
Originally developed for the Universidade Federal do Triângulo Mineiro (UFTM) to comply with the Brazilian Marco Civil da Internet (Law 12.965/2014) requirements for network log retention.
